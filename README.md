# Docker C

linux 下 C 语言开发构建环境

![Travis (.com)](https://img.shields.io/travis/com/alvisisme/docker-c)
![Docker Pulls](https://img.shields.io/docker/pulls/alvisisme/linux-c)

## 拉取镜像

```bash
docker pull alvisisme/linux-c
```

## 工具链说明

* 编译器 gcc
* 构建工具 autotools make cmake
* 代码风格 splint
* 文档 doxygen
* 单元测试 cmockery
* 单元测试覆盖率 gcovr
